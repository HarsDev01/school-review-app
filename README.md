## Features
- Add and view school reviews
- MySQL integration
- Simple Bootstrap UI

## Setup
1. Install dependencies  
   `pip install -r requirements.txt`

2. Set up MySQL DB using `reviews.sql`

3. Add your DB credentials to `.env`

4. Run the app  
   `python app.py`
